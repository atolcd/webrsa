La conversion du fichier
Le publipostage
  1. La conversion du fichier .csv

    2. Télécharger le tableau depuis webrsa en cliquant sur le bouton.

    Si vous utilisez La suite bureautique OpenOffice, ouvrez le fichier avec Calc.

    Double cliquer depuis la fenêtre de téléchargement.

    Choisir l'option d'ouverture avec Open Office.

    Valider en cliquant sur "OK"  :

    Les données apparaissent.

    Enregistrer le fichier au format Open Office (.ods) pour pouvoir l'utiliser pour le publipostage.



  2. Le publipostage

    3. Ouvrir la lettre type enregistrée au format Open Office (.odt)

    NB : La source de données à fusionner avec le fichier doit également être au format Open Office (en principe un fichier .ods).

    Menu Outils / Assistant mailing.

    L'assistant permet de lier la source des données.

    La touche F4 affiche la source des données liées.

    Valider les étapes de l'Assistant en cliquant sur le bouton "Suivant".

    Cliquer sur l'étape de sélection de la source de données  :

    Sélectionner le fichier Open Office issu de webrsa.

    Valider en cliquant sur le bouton OK.

    A ce stade, il est possible de fermer l'assistant, de récupérer la source de données en appuyant sur la touche F4, puis d'insérer les champs de fusion dans le document en effectuant un cliqué-glissé des en-têtes de colonnes de nos données.

    On peut aussi continuer avec l'assistant et faire correspondre les champs  :

    Cocher la case : "Ce document contiendra un bloc d'adresses", puis cliquer sur le bouton "Correspondance des champs".

    Valider les correspondances des champs avec OK.

    Désactiver les salutations.

    Désactiver le bloc d'adresse : Le bloc d'adresse insère les champs de fusion à l'endroit où le curseur est positionné dans le document; Pour insérer les champs de fusion à des endroits différents dans le document, on peut afficher la source de données (F4).

    Il est possible de positionner le bloc d'adresse en précisant l'alignement :   "A partir de..."

    On peut pré-visualiser le document en cliquant sur le bouton "Editer le document…".

    Le bouton "Editer" le document permet de contrôler les enregistrements.

    Un bouton s'affiche dans le document qui vous permet de revenir à l'assistant, depuis l'assistant, cliquer sur le bouton "Suivant" pour générer les courriers.

    Terminer le publipostage en choisissant l'option de fusion pour l'enregistrement des courriers, puis enregistrer.

    On simplifie la procédure :

    Autre méthode (plus simple): Appuyer sur la touche F4 pour afficher la source de données liées.

    Cliquer / Glisser les champs de la base de données dans le document aux endroits de l'insertion.

    On insère les champs à leur emplacement par un cliquer-glisser des en-têtes de colonne dans le document.

    Menu Outils / Assistant mailing.

    Se positionner directement sur la dernière étape de la fusion.

    Les courriers sont générés automatiquement.

    Enregistrer les documents fusionnés.
